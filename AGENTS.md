# Repository Guidelines

## General rules

All text on the pages is in German by default. There is a UI control (button, link, or the like) that can be used to switch to English.

## Technologies Used

- The website should be developed with the latest Angular version.
- Styling should be done using Tailwind.
- For component / unit testing Vitetest should be used.
- For end-to-end testing Playwright is selected.

## Site Map

Build a new JOVISCO website, consisting of:

- a one-pager that contains the main content,
- a separate Contact page (Kontakt in German),
- a separate Legal Notice page (Impressum in German).

## Logos and Colour

The website should exist with 4 different color schemes:

- original: as found in file assets/img/originals/jovisco_logo_183x50.png and ./assets/img/originals/jovisco_logo-solo.png
- dark: a dark scheme with a almost black background, a dark red accent color and a dark grey color.
- light: same color scheme as original
- metal: a blueish scheme which looks like metallic.

Fitting logos must be created for the different color scheme.
